errno is a thread local variable, that means it is like a global variable (thus accessible anywhere) but each thread has its own value. To do that errno is a define that is replaced by *__errno_location() __errno_location() gives an address different for each thread *__errno_location() dereferences the pointeur, i.e. follows the pointer and it designates what is pointed to. #define errno *__errno_location()

errno_mess contains the error messages for each error code this table is define in both files libc.c and klib.c but should be identical
