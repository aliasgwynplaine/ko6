7 \file common/debug_on.h
8 \author Franck Wajsburt
9 \brief Debug messages
10 Permanent
11 - PANIC_IF( cond, fmt, arg...) to stop the execution in case of a fatal condition
12 Temporary
13 - Macros BIP(), VAR(), VARN(), INFO() & ASSERT() adds temporary debug messages
14 - This is useful to find and fix errors,
15 but we don't want to display messages if there is no bugs
16 - Add #include <degug_on.h> before the messages you want to display
17 so that the BIP(), VAR(), VARN(), INFO() and ASSERT() macros produce messages.
18 - Add #include <degug_off.h> to stop messages from BIP(), INFO() and so on.
19 - It is possible to include these files several times per c file
