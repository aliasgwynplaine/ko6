Detailed Description

Simple fifo (1 writer - 1 reader)

  • data buffer of data
  • pt_write write pointer for L fifos (0 at the beginning)
  • pt_read read pointer for L fifos (0 at the beginning)

data[] is used as a circular array. At the beginning (pt_read == pt_write) means an empty fifo then when we push a data, we write it at pt_write, the we increment pt_write % fifo_size. The fifo is full when it remains only one free cell, then when (pt_write + 1)size == pt_read
